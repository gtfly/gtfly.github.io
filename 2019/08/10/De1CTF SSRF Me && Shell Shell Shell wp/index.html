geneSign路由的action恒为scan，那么当param=flag.txt时，getSign为：

md5(secret_key+'flag.txt'+'scan')

那么要使read出现在action中，可以添加在flag.txt后，便可获取其hash：

md5(secret_key+'flag.txtread'+'scan')

在De1ta路由中，由于读取的是param中的内容，便可将flag.txt与read分隔开，使action的值为readscan，即：

md5(secret_key+'flag.txtread'+'scan') ==> md5(secret_key+'flag.txt'+'readscan')

    }

之后取数组的最后一个元素，作为$ext：

进行判断

if($ext==$filename[count($filename) - 1]){

    }

正常上传一个文件名为a.txt的文件，那么$filename会为：

[0]=>'a'
[1]=>'txt'

count($filename)-1等于1，那么$filename[count($filename) - 1]为txt，$ext为txt，则两者相等，因此需要想办法绕过判断

可以使$filename数组中最后一个元素的键为非连续的值，从而使$filename[count($filename) - 1]为空，因为有这句话：

$filename = !empty($_POST['file']) ? $_POST['file'] : $_FILES['file']['name'];

因此可以在上传的同时POST值：

file[0]=a
file['a']=php

这样，$filename就等于上面的两个元素，$ext为php，$filename[count($filename) - 1]为空，这样便绕过了第一个判断
